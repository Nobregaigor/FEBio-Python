Compare Pressure and Volume Results¶

-- Info¶

This notebook can be used to compare the results obtained from a FEBio simulation and the ideal volume curve

Results from simulation must be in a pickle dataframe with "volume" and "timestep" columns. The "PAQ-GammaStudy.ipynb" can be used to compute this dataframe. PV-curve must be in a pickle dataframe. The "smooth_data.ipynb" can be used to comput this dataframe.

# define an interpolation wrapper
from scipy.interpolate import interp1d
def interp_wrap(x,y,x_new, kind='linear'):
    f = interp1d(x, y, kind=kind)
    y_new = f(x_new)
    return y_new.reshape((y_new.shape[0], 1))
